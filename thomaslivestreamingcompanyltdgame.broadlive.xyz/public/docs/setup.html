🚀 Server Setup Instructions

Requirements

  • PHP 8.1 or higher
  • MySQL or MariaDB
  • Composer
  • Web server (Apache/Nginx) with document root pointing to the Laravel app

Step 1: Environment Setup

  1. Copy .env.example to .env
  2. Edit .env with your settings: 
    APP_NAME="Thomas Game"
APP_URL=https://fairylive.online

DB_CONNECTION=mysql
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=play
DB_USERNAME=root
DB_PASSWORD=your_password

APP_KEY=base64:xxxxxxxxxxxxxxxxxxxxx

Step 2: Install Dependencies

composer install --no-dev --optimize-autoloader

Step 3: Generate Application Key

php artisan key:generate

Step 4: Database Setup

Option A: Import from database dump

mysql -u root -p play < database/dumps/thomas-game.sql

Option B: Run migrations

php artisan migrate

Step 5: Clear Cache

php artisan optimize:clear

Step 6: Start Game Round Loop

This process handles game rounds and settlements:

php artisan bdgamefinal:watch --sleep=1
💡 Note: The bdgamefinal:watch command should run continuously. Use a process manager like Supervisor or PM2 to keep it running.

Supervisor Configuration (Recommended)

Create /etc/supervisor/conf.d/thomas-game.conf:

[program:thomas-game-worker]
process_name=%(program_name)s_%(process_num)02d
command=php /path/to/play/artisan bdgamefinal:watch --sleep=1
autostart=true
autorestart=true
numprocs=1
redirect_stderr=true
stdout_logfile=/path/to/play/storage/logs/worker.log

Then reload supervisor:

sudo supervisorctl reread
sudo supervisorctl update
sudo supervisorctl start thomas-game-worker:*

Web Server Configuration

Apache (.htaccess): Usually already configured

Nginx:

server {
    listen 80;
    server_name fairylive.online;
    root /path/to/play/public;
    
    index index.php;
    
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }
    
    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
    }
}
⚠️ Important: Always use HTTPS in production. Install an SSL certificate using Let's Encrypt.

Verification

  1. Visit your game lobby: https://yoursite.com/play_bd_game
  2. Check logs: storage/logs/laravel.log
  3. Test API: https://yoursite.com/game/api/lucky88_master/status

🚀 সার্ভার সেটআপ নির্দেশাবলী

প্রয়োজনীয়তা

  • PHP 8.1 বা উচ্চতর
  • MySQL বা MariaDB
  • Composer
  • ওয়েব সার্ভার (Apache/Nginx) Laravel অ্যাপের সাথে সংযুক্ত

ধাপ ১: পরিবেশ সেটআপ

  1. .env.example থেকে .env কে কপি করুন
  2. .env সম্পাদনা করুন আপনার সেটিংস দিয়ে: 
    APP_NAME="Thomas Game"
APP_URL=https://fairylive.online

DB_CONNECTION=mysql
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=play
DB_USERNAME=root
DB_PASSWORD=আপনার_পাসওয়ার্ড

APP_KEY=base64:xxxxxxxxxxxxxxxxxxxxx

ধাপ ২: নির্ভরতা ইনস্টল করুন

composer install --no-dev --optimize-autoloader

ধাপ ৩: অ্যাপ্লিকেশন কী তৈরি করুন

php artisan key:generate

ধাপ ৪: ডাটাবেস সেটআপ

বিকল্প A: ডাটাবেস ডাম্প থেকে আমদানি করুন

mysql -u root -p play < database/dumps/thomas-game.sql

বিকল্প B: মাইগ্রেশন চালান

php artisan migrate

ধাপ ৫: ক্যাশ সাফ করুন

php artisan optimize:clear

ধাপ ৬: গেম রাউন্ড লুপ শুরু করুন

এই প্রক্রিয়াটি গেম রাউন্ড এবং নিষ্পত্তি পরিচালনা করে:

php artisan bdgamefinal:watch --sleep=1
💡 নোট: bdgamefinal:watch কমান্ডটি ক্রমাগত চলবে। এটি চালু রাখতে Supervisor বা PM2 ব্যবহার করুন।

Supervisor কনফিগারেশন (সুপারিশকৃত)

/etc/supervisor/conf.d/thomas-game.conf তৈরি করুন:

[program:thomas-game-worker]
process_name=%(program_name)s_%(process_num)02d
command=php /পথ/থেকে/play/artisan bdgamefinal:watch --sleep=1
autostart=true
autorestart=true
numprocs=1
redirect_stderr=true
stdout_logfile=/পথ/থেকে/play/storage/logs/worker.log

তারপর supervisor পুনরায় লোড করুন:

sudo supervisorctl reread
sudo supervisorctl update
sudo supervisorctl start thomas-game-worker:*

ওয়েব সার্ভার কনফিগারেশন

Apache (.htaccess): সাধারণত ইতিমধ্যে কনফিগার করা থাকে

Nginx:

server {
    listen 80;
    server_name fairylive.online;
    root /পথ/থেকে/play/public;
    
    index index.php;
    
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }
    
    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
    }
}
⚠️ গুরুত্বপূর্ণ: উৎপাদনে সর্বদা HTTPS ব্যবহার করুন। Let's Encrypt ব্যবহার করে SSL সার্টিফিকেট ইনস্টল করুন।
