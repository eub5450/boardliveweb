1) Login ও Security Pass

অ্যাডমিনে ঢোকার আগে দুই ধাপ থাকে। প্রথমে admin login, তারপর game security pass।

Two-step access
1
Email + Password /thomas-admin এ গিয়ে admin email/password দিন। ভুল login হলে lock হতে পারে।
2
Security Pass Login এর পর security page এ pass বসিয়ে dashboard খুলতে হবে। এটা extra protection layer.
3
Admin Session একবার access পেলে games, rounds, bets, payouts, users সব দেখতে পারবেন।

2) Dashboard কীভাবে কাজ করে

Dashboard হলো control room. এখান থেকে game status, maintenance, visible room, payout, এবং room-level settings নিয়ন্ত্রণ হয়।

Main control room
A
Game On / Off / Maintenance একটি game live থাকবে, hide হবে, অথবা maintenance mode এ যাবে। Maintenance হলে developer ID whitelist কাজ করতে পারে।
B
Board Payout Control প্রতিটি board-এর payout multiplier edit করা যায়। সাধারণত active mode-এ careful change করতে হয়।
C
Lobby Visibility কোন game lobby-তে দেখাবে, কোনটা hidden থাকবে, সেটাও এখান থেকেই control হয়।

5) Common Tasks

অ্যাডমিনরা সাধারণত এই কাজগুলো করেন।

Practical guide
1
গেম maintenance এ রাখা Dashboard এ game status maintenance করুন, তারপর প্রয়োজন হলে নির্দিষ্ট developer ID allow করুন।
2
ইন-অ্যাকটিভ user refresh Network off, phone sleep, বা lock হলে session refresh / re-sync দরকার হতে পারে। Backend flow থাকলে সেটাই active থাকে।
3
Wallet review Users page থেকে wallet transfer, lock, profile open করা যায়। সবসময় journal entry যাচাই করুন।
4
Round audit Rounds ট্যাবে গিয়ে winner, time, and decision mode দেখে settlement ঠিক কিনা মিলিয়ে নিন।

6) Mobile ও Troubleshooting

ছোট স্ক্রিনে কাজ করার সময় যেসব জিনিস খেয়াল করবেন।

Mobile safe
  • Mobile-এ table long হলে horizontal scroll ব্যবহার করুন, fullscreen table দেখার চেষ্টা করবেন না।
  • Density toggle থাকলে Compact mode long rows পড়তে সাহায্য করে।
  • Security pass ভুল দিলে login/session lock হতে পারে, তাই বারবার ভুল চেষ্টা না করাই ভালো।
  • Maintenance mode এ শুধু অনুমোদিত developer ID-ই game play করতে পারবে।
  • Round বা payout edit করার আগে relevant game live আছে কিনা যাচাই করুন।

এই ম্যানুয়ালটা নতুন operator বা support team-এর জন্য বানানো। চাইলে আমরা পরে এর একটা আরও ছোট “one-page cheat sheet” version বানাতে পারি, যেটা mobile-এ দ্রুত পড়া যাবে।
